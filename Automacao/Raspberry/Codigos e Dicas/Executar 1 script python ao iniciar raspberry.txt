Abra o terminal e digite


sudo nano crontab -e

vamos usar como exemplo um arquivo que esta dentro de /home/pi e que se chame led.py


Para configurar a raspberry pra iniciar ja executando este script basta adicionar no final do arquivo do crontab a seguinte linha


@reboot python3 /home/pi/led.py &

ATENCAO: Nao esquecer do & no fim do comando

Aperte Control+X em seguida Y para confirmar e de enter para salvar

Sudo reboot para reiniciar e esta pronto!
