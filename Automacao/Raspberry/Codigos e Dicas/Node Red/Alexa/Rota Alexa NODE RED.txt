Para criar o roteamente para a alexa funcionar no node rede, vamor redirecinar a porta 80 para 8080

Abra o terminal e digite

sudo nano /etc/init.d/port

******************************************************

#! /bin/sh
# /etv/init.d/port
#

# Some things that run always

 sudo iptables -I INPUT 1 -p tcp --dport 80 -j ACCEPT
 sudo iptables -A PREROUTING -t nat -i eth0 -p tcp --dport 80 -j REDIRECT --to-port 8080

exit 0

*******************************************************

Dê um control+X em seguida confirme com S e enter

update-rc.d port defaults

confirme 1 para usuario pi e coloque a senha atual ( nesse caso, raspberry )

Deve aparecer Authentication Complete

Agora vamos confirugrar o sistema ao iniciar chamar este script

Digite

>crontab -e

Confirme 1 para nano

Insira no fim do arquivo
@reboot /etc/init.d/port start

Dê um control+X e S e enter em seguida para salvar


Agora está pronto. Basta dar um sudo reboot
