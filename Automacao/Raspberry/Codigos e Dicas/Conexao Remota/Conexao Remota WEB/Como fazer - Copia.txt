abra o terminal e digite

> git clone https://github.com/novnc/noVNC.git e dê enter

ao acabar digite
> cd noVNC e dê enter

em seguida 
> ls e dê enter para listar a pasta

em seguida digite

> ./utils/launcher.sh --vnc localhost:5900 e dê enter

irá aparecer a url para acessar, não preocupe agora. Dê um Control+C para cancelar

ATENÇÃO:
Faça esta parte senão não irá funcionar

Conececte via VNC normal ou via uma tela e clica com botão direito no icone do VNC na barra em cima perto do ícone do bluetooth e clicar em opções. Irá abrir uma tela e em segurança 

Criptografia: Colocar Preferir desativada
e em 
Autenticacao : Colocar Senha do VNC e clicar em Aplicar, irá abrir uma tela e digite a senha e confirme novamente

Agora vamos habilitar para iniciar automáticamente no reboot

Digite por segurança cd cd cd
agora

> ps aux | grep web e dê enter

agora digite

> sudo nano /etc/systemd/system/noVnc.service e dê enter, irá abrir uma tela em branco

coloque os dados abaixo

**************************************************************************

[Unit]
Description=noVnc
After=graphical.target

[Service]
Type=Simple
User=pi
ExecStart=/home/pi/noVNC/utils/launch.sh --vnc localhost:5900 > /dev/null 2>&1 &

[Install]
WantedBy=graphical.target


******************************************************************
Agora dê Control+X e S para confirmar e dê enter para salvar

Agora dê o comando

> sudo systemctl daemon-reload e dê enter

Em seguida reiniciamos o serviço criado

> sudo systemctl restart noVnc.service e dê enter

Agora verificando se deu tudo certo, digite o comando abaixo

>sudo systemctl status noVnc.service

OBS: Deve retornar como "Active!"

Agora devemos habilitar o serviço com o seguinte comando

> sudo systemctl enable noVnc.service

Para finalizar basta dar um sudo reboot para reinicar!


Para acessar via web basta agora abrir a URL

>>>>> http://IP_RASPBERRY:6080/vnc.html?host=IP_RASPBERRY&port=6080
