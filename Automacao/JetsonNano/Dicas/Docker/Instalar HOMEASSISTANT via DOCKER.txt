Adicionando o homeassistant container

Abra o site "https://www.home-assistant.io/installation/raspberrypi" e procure a instalacao via docker, abaixo tera o codigo proximo a este abaixo aqui

docker run -d \
  --name homeassistant \
  --privileged \
  --restart=unless-stopped \
  -e TZ=America/Sao_Paulo \
  -v /root/homeassistant/config:/config \
  --network=host \
  ghcr.io/home-assistant/home-assistant:stable



docker run -d \
  --name homeassistant2 \
  --privileged \
  --restart=unless-stopped \
  -e TZ=America/Sao_Paulo \
  -v /root/homeassistant2/config:/config \
  --network=host \
  ghcr.io/home-assistant/home-assistant:latest
