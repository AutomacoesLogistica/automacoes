Abra o cmd e digite

Instale o netplan.io
>sudo apt-get install -y netplan.io

Agora vamos criar o profile
> sudo touch /etc/netplan/01-ethernet.yaml


Aora editar o arquivo de configuracoes de rede
> sudo nano /etc/netplan/01-ethernet.yaml

OBS: para esse exemplo iremos usar ip: 192.168.10.35 com gateway 192.168.10.78

********************************************
network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      addresses:
        - 192.168.10.35/24
      gateway4: 192.168.10.78
      nameservers:
        addresses:
          - "192.168.10.78"
          - "8.8.8.8"

********************************************

DÃª um control +x e confirma para salvar

Agora vamos confirmar e aplicar os dados configurados

> sudo netplan apply

Agora podemos verificar se deu tudo certo com o comando abaixo

> sudo ip addr show dev eth0
